1. Shape файл должен содержать полилинии,
	координаты точек которых соответствуют
	широте и долготе в десятичных градусах WGS-84
	EPSG:4326 / SR-ORG:14 / WGS 84
	
	Каждая первая/последняя точка полилинии является
	узлом графа, каждая полилиния - ребром
	
2. Кодировка DBF-файла должна быть Windows-1251 или DOS-866

3. Обязательные поля DBF-файла:
	
	LINK_ID - уникальный целочисленный идентификатор 
		линии в файле, используется для запретов 
		поворотов и связи искомого shape-файла с графом
		
	GRMN_TYPE - тип дороги по классификатору Garmin
	
	ROUTE_SPEED - скорость движения на участке в км/ч
	
	SPD_LIMIT - ограничение максимальной скорости в км/ч
	
	ONE_WAY - признак одностороннего движения (0/1)
	
	TURN_RSTRS - запреты поворотов в формате F0000;L0000
		где F/L - запрет поворота из первой/последней точки
		линии в LINK_ID 0000
		
	LEN - длина полилинии в метрах
	
	TMC_CODES1 - Поле с информацией о TMC коде в формате
		@E0+000+00000@E0-000-00000
		@E0+000+00000;@E0-000-00000;
		@E0+000+00000 или @E0-000-00000 (TMC сонаправлен с линией)
        @E0+000-00000 или @E0-000+00000 (ТМС противоположенно направлен линии)
		
	RGNODE - Поле с номером точки стыковки межрайонных маршрутов
		формат: F0001IO / L0002I / F0003O
		где: F/L - превая или последняя точка линии - необходимый узел
		0000 - идентификатор межрайонного узла
		I/O/IO - I (In) - через этот узел можно въехать в регион
			- O (out) - через этот узел можно выехать из региона
	
	NAME - Поле с наименованием линии
	
	ATTR - Поле с атрибутами дорог (список кодов через запятую)	
	ML - ограничение массы в тоннах
	AL - максимальная нагрузка на ось в тоннах
	HL - ограничение высоты в метрах
	WL - ограничение ширины в метрах
	LL - ограничение длины в метрах
	DL - минимальная дистанция между ТС в метрах

4. Для настройки соответствия полей необходимо создать файл с именем
	shape файла, но с расширением .fldcfg.xml, из которого будет взята
	вся конфигурация (например, lipets.shp -> lipets.fldcfg.xml)
	
	SAMPLE.fldcfg.xml //  default.fldcfg.xml
	<!----------------------- BEGIN ------------------------>
		<?xml version="1.0" encoding="utf-8"?>
			<ShapeFields version="20160331L">
				<SOURCE>GARMIN</SOURCE>
				<CodePageId>1251</CodePageId>
				<LinkId>LINK_ID</LinkId>
				<GarminType>GRMN_TYPE</GarminType>
				<RouteSpeed>ROUTE_SPEED</RouteSpeed>
				<SpeedLimit>SPD_LIMIT</SpeedLimit>
				<OneWay>ONE_WAY</OneWay>
				<RouteLevel>ROUTE_LVL</RouteLevel>
				<TurnRestrictions>TURN_RSTRS</TurnRestrictions>
				<Length>SHAPE_LENG</Length>
				<TMC>TMC_CODES1</TMC>
				<RGNODE>RGNODE</RGNODE>
				<NAME>STRT_ADDR</NAME>
				<ACC_MASK>ACC_MASK</ACC_MASK>
				<ATTR>ATTR</ATTR>
				<MaxWeight>ML</MaxWeight>
				<MaxAxle>AL</MaxAxle>
				<MaxHeight>HL</MaxHeight>
				<MaxWidth>WL</MaxWidth>
				<MaxLength>LL</MaxLength>
				<MinDistance>DL</MinDistance>
			</ShapeFields>
    <!------------------------ END ------------------------->
